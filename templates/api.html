Описание API для сайта "Что в кино"

API предоставляет демонстрационные запросы возвращающие агрегированную информацию с сайтов Афиша, Кинопоиск и TMDB

  • /api/movies - вернет описание для фильмов представленных на главной странице в формате JSON
  • /api/movies/:result_amount/cinemas/:cinemas_amount - вернет аналогичный предыдущему ответ с возможностью выбора максимального кол-ва строк результата и изменения значения фильтра по кол-ву кинотеатров в которых в данный момент показывается фильм
    • :result_amount - максимальное количество результатов
    • :cinemas_amount - фильтр по минимальному количеству кинотеатров в которых идет показ фильма по данным Афиша.ру
